Arrays y ArrayList

En esta sección aprenderás a manejar colecciones de datos en Java. Los arrays son estructuras de tamaño fijo que almacenan elementos del mismo tipo, mientras que ArrayList es una clase de la API de Java que permite crear listas dinámicas (redimensionables).

Conceptos clave

  • Array: Estructura de tamaño fijo, se declara con corchetes []. Puede ser unidimensional o multidimensional.
  • ArrayList: Implementación de la interfaz List que puede crecer y reducirse automáticamente. Solo almacena objetos (no tipos primitivos directamente, usa clases envolventes). Es una lista de objetos de tipo genérico.

Arrays Unidimensionales

Un array unidimensional es una secuencia de elementos del mismo tipo accesibles mediante un índice (que comienza en 0).

Los arrays tienen un tamaño fijo que se define al crearlos. No se puede cambiar su tamaño una vez creado.

Declaración y creación

// Declaración
int[] numeros;          // más común
String nombres[];       // también válido, pero menos usado

// Creación (con tamaño fijo)
numeros = new int[5];   // array de 5 enteros (inicializados a 0)

// Declaración y creación en una línea
int[] edades = new int[10];

// Inicialización directa con valores
int[] valores = {10, 20, 30, 40, 50};

Acceso y modificación

int[] numeros = new int[3];
numeros[0] = 100;
numeros[1] = 200;
numeros[2] = 300;
System.out.println(numeros[1]); // Imprime 200
Cuidado: Cuando se hace una inicialización directa con valores, el tamaño del array se determina automáticamente. No puedes agregar más elementos de los que se han definido en la inicialización. Si necesitas una colección de tamaño variable, considera usar ArrayList.

Recorrer un array

// Con bucle for clásico
for (int i = 0; i < numeros.length; i++) {
    System.out.println("Índice " + i + ": " + numeros[i]);
}

// Con bucle foreach (desde Java 5)
for (int num : numeros) {
    System.out.println("Valor: " + num);
}
Cuidado: Los arrays tienen tamaño fijo. Si intentas acceder a un índice fuera del rango (menor que 0 o mayor o igual a length) se lanzará una excepción ArrayIndexOutOfBoundsException.

Arrays Multidimensionales

Son arrays de arrays, útiles para representar matrices, tablas, etc. Se declaran con dos o más pares de corchetes.

Ejemplo con matriz de 2x3

// Declaración y creación
int[][] matriz = new int[2][3];  // 2 filas, 3 columnas

// Asignación de valores
matriz[0][0] = 1;
matriz[0][1] = 2;
matriz[0][2] = 3;
matriz[1][0] = 4;
matriz[1][1] = 5;
matriz[1][2] = 6;

// Inicialización directa
int[][] matriz2 = {
    {1, 2, 3},
    {4, 5, 6}
};

// Recorrer una matriz
for (int i = 0; i < matriz.length; i++) {          // filas
    for (int j = 0; j < matriz[i].length; j++) {   // columnas
        System.out.print(matriz[i][j] + " ");
    }
    System.out.println(); // salto de línea al final de cada fila
}

El ejemplo anterior recorre una matriz bidimensional e imprime cada elemento seguido de un espacio. Al final de cada fila, se imprime un salto de línea para mantener una estructura legible.

Quedando así una representación visual de la matriz:

1 2 3 
4 5 6

La clase ArrayList

ArrayList es una clase del paquete java.util que proporciona una lista redimensionable. A diferencia de los arrays, puede crecer o encogerse dinámicamente.

Pasos para usar ArrayList:

  1. Importar la clase: import java.util.ArrayList;
  2. Crear el ArrayList: ArrayList<Tipo> lista = new ArrayList<>();
  3. Usar sus métodos para agregar, eliminar, acceder, etc.

Métodos principales de ArrayList

Método Descripción Ejemplo
add(elemento) Añade un elemento al final de la lista. lista.add("Hola");
add(índice, elemento) Inserta un elemento en la posición indicada. lista.add(1, "Mundo");
get(índice) Devuelve el elemento en la posición especificada. String s = lista.get(0);
set(índice, elemento) Reemplaza el elemento en la posición dada. lista.set(0, "Adiós");
remove(índice) Elimina el elemento en la posición indicada. lista.remove(1);
remove(elemento) Elimina la primera ocurrencia del elemento especificado. lista.remove("Hola");
size() Devuelve el número de elementos de la lista. int tam = lista.size();
clear() Elimina todos los elementos. lista.clear();
isEmpty() Devuelve true si la lista está vacía. boolean vacia = lista.isEmpty();
contains(elemento) Devuelve true si la lista contiene el elemento. boolean tiene = lista.contains("Hola");
indexOf(elemento) Devuelve el índice de la primera aparición del elemento, o -1 si no está. int idx = lista.indexOf("Mundo");

Ejemplo completo con ArrayList

import java.util.ArrayList;

public class EjemploArrayList {
    public static void main(String[] args) {
        // Crear un ArrayList de Strings
        ArrayList<String> frutas = new ArrayList<>();

        // Agregar elementos
        frutas.add("Manzana");
        frutas.add("Banana");
        frutas.add("Cereza");
        frutas.add(1, "Naranja"); // Insertar en la posición 1

        System.out.println("Lista de frutas: " + frutas);

        // Acceder a un elemento
        String fruta = frutas.get(2);
        System.out.println("Fruta en índice 2: " + fruta);

        // Modificar un elemento
        frutas.set(0, "Pera");
        System.out.println("Después de modificar: " + frutas);

        // Eliminar un elemento
        frutas.remove(3); // Elimina "Cereza"
        frutas.remove("Banana"); // Elimina "Banana"
        System.out.println("Después de eliminar: " + frutas);

        // Tamaño de la lista
        System.out.println("Tamaño: " + frutas.size());

        // Recorrer con foreach
        for (String f : frutas) {
            System.out.println("Fruta: " + f);
        }

        // Comprobar si contiene un elemento
        if (frutas.contains("Pera")) {
            System.out.println("Sí, hay Pera");
        }

        // Vaciar la lista
        frutas.clear();
        System.out.println("¿Está vacía? " + frutas.isEmpty());
    }
}

En este ejemplo se muestra cómo crear un ArrayList de objetos de tipo String, agregar elementos, acceder a ellos, modificarlos, eliminarlos y recorrer la lista. También se demuestra cómo verificar si la lista contiene un elemento específico y cómo vaciar la lista por completo.

Crea la lista de frutas, agrega elementos, modifica uno de ellos y elimina otro. Finalmente, comprueba si existe el elemento "Pera" y borra la lista para verificar si está vacía.

Nota sobre tipos primitivos: ArrayList no puede almacenar tipos primitivos directamente (int, double, etc.). Debes usar las clases envolventes: Integer, Double, Boolean, etc. Java realiza autoboxing y unboxing automáticamente. 
ArrayList<Integer> numeros = new ArrayList<>();
numeros.add(10); // autoboxing de int a Integer
int valor = numeros.get(0); // unboxing de Integer a int

Métodos útiles para arrays y ArrayList

Java proporciona clases auxiliares como Arrays y Collections con métodos estáticos muy prácticos.

Clase java.util.Arrays

Método Descripción Ejemplo
toString(array) Devuelve una representación en cadena del array. System.out.println(Arrays.toString(numeros));
sort(array) Ordena el array en orden ascendente. Arrays.sort(numeros);
fill(array, valor) Rellena todo el array con el valor especificado. Arrays.fill(numeros, 0);
copyOf(array, nuevaLongitud) Copia el array con la nueva longitud. int[] copia = Arrays.copyOf(original, 10);

Clase java.util.Collections para ArrayList

Método Descripción Ejemplo
sort(lista) Ordena la lista (sus elementos deben implementar Comparable). Collections.sort(lista);
reverse(lista) Invierte el orden de la lista. Collections.reverse(lista);
shuffle(lista) Desordena aleatoriamente la lista. Collections.shuffle(lista);
max(lista) Devuelve el elemento máximo según el orden natural. String max = Collections.max(lista);
min(lista) Devuelve el elemento mínimo. String min = Collections.min(lista); 
import java.util.Arrays;
import java.util.ArrayList;
import java.util.Collections;

public class EjemploMetodosUtiles {
    public static void main(String[] args) {
        // Con arrays
        int[] numeros = {5, 2, 9, 1, 3};
        Arrays.sort(numeros);
        System.out.println(Arrays.toString(numeros)); // [1, 2, 3, 5, 9]

        // Con ArrayList
        ArrayList<String> nombres = new ArrayList<>();
        nombres.add("Juan");
        nombres.add("Ana");
        nombres.add("Pedro");
        Collections.sort(nombres);
        System.out.println(nombres); // [Ana, Juan, Pedro]
    }
}

En este ejemplo se muestra cómo usar métodos de las clases Arrays y Collections para ordenar, imprimir y manipular arrays y ArrayLists de manera eficiente.
