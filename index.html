Aprendiendo Java, paso a paso

Desde cero con Oracle JDK 21 y Eclipse

¡Bienvenido!

Esta web te guiará en tu camino para aprender Java desde cero. Utilizaremos la versión LTS, Oracle JDK 21 , y el entorno de desarrollo integrado (IDE) Eclipse.

Conceptos Clave Iniciales

  • Workspace (Espacio de trabajo): Es la carpeta en tu ordenador donde Eclipse guarda todos tus proyectos, configuraciones y preferencias. Piensa en él como tu "oficina" de programación.
  • Proyecto: Es una carpeta dentro del workspace que contiene todos los archivos necesarios para una aplicación específica (código, imágenes, librerías).
  • Package (Paquete): Es una carpeta dentro del proyecto que usamos para organizar nuestras clases de Java de forma lógica (por ejemplo, por funcionalidad). Sus nombres suelen ser en minúsculas y con dominios invertidos (ej: `com.miweb.util`).

Configurando el Entorno: Eclipse

Aquí tienes los pasos esenciales para crear tu primer proyecto en Eclipse. Al escribir un nombre recuerda que distingue entre mayúsculas y minúsculas (Case Sensitive)!


  1. Crear un nuevo Proyecto Java: Ve a File -> New -> Java Project. Dale un nombre (ej. `MiPrimerProyecto`). Asegúrate de que la JRE seleccionada sea la de Java 21. Haz clic en "Finish".
  2. Crear un Package: Dentro de la carpeta `src` de tu proyecto, haz clic derecho y selecciona New -> Package. Nómbralo (ej. `misprimerosprogramas`). Es una buena práctica crear siempre un paquete para organizar el código.
  3. Crear una Clase: Haz clic derecho en el package que acabas de crear y selecciona New -> Class. Así es donde escribiremos nuestro código Java.

El Primer Proyecto: Hola Mundo

Sigue estos pasos para crear tu primer programa. Recuerda, el nombre de la clase debe coincidir EXACTAMENTE con el nombre del archivo.

Pasos en Eclipse

  1. Crea un nuevo proyecto llamado `ProyectoHolaMundo`.
  2. Dentro de `src`, crea un package `holamundo`.
  3. Dentro del package, crea una nueva clase llamada `HolaMundo`.
  4. Copia el siguiente código en el editor, DENTRO de las llaves de la clase `HolaMundo` que Eclipse ha creado para ti:
public static void main(String[] args) {
    // Esto es un comentario. El programa empezará aquí.
    System.out.println("¡Hola, Mundo!");
}

Para ejecutar: Haz clic derecho sobre el archivo `HolaMundo.java` en el explorador de proyectos, ve a Run As -> Java Application. Verás el mensaje en la consola.

Importante: Fíjate que `main` va en minúsculas, `String` con mayúscula, y `System` con mayúscula. ¡Un error de mayúscula/minúscula impedirá que tu programa funcione!

Tipos de Datos Primitivos en Java

Java tiene tipos de datos fundamentales. Aquí los más usados:

Tipo Descripción Ejemplo Tamaño
int Números enteros int edad = 30; 32 bits
double Números decimales de doble precisión double precio = 19.99; 64 bits
boolean Verdadero o falso boolean esMayor = true; 1 bit
char Un solo carácter (con comillas simples) char inicial = 'A'; 16 bits

Ejemplo de uso:

int anio = 2024;
double pi = 3.14159;
boolean cursoActivo = true;
char letra = 'J';

System.out.println("Estamos en " + anio + ". El valor de pi es " + pi);

Tipos de Datos: String

El tipo de dato String se usa para almacenar cadenas de texto. A diferencia de otros tipos primitivos, String no es primitivo, sino una clase en Java.

String nombre = "Juan";
String mensaje = "Hola, " + nombre + "!";

String tiene muchos métodos útiles. Aquí los más usados:

Método Descripción Ejemplo
length() Devuelve la longitud de la cadena int len = nombre.length();
toUpperCase() Convierte la cadena a mayúsculas String mayus = nombre.toUpperCase();
toLowerCase() Convierte la cadena a minúsculas String minus = nombre.toLowerCase();
substring(int start, int end) Obtiene una subcadena desde 'start' hasta 'end' String sub = nombre.substring(0, 2); // "Ju"
contains(String str) Verifica si la cadena contiene otra cadena boolean tieneA = nombre.contains("a"); // true
replace(String target, String replacement) Reemplaza todas las ocurrencias de 'target' por 'replacement' String nuevo = nombre.replace("a", "o"); // "Jun"
trim() Elimina espacios en blanco al inicio y al final String limpio = " Hola ".trim(); // "Hola"
equals(Object obj) Compara dos cadenas para ver si son iguales boolean igual = nombre.equals("Juan"); // true
charAt(int index) Devuelve el carácter en la posición especificada char c = nombre.charAt(0); // "J"

Obteniendo Datos: La Clase Scanner

Para que nuestros programas interactúen con el usuario, necesitamos leer lo que escribe por teclado. Usamos la clase `Scanner`.

Pasos para usar Scanner:

  1. Importar la clase: Añade al principio del archivo (justo después del package): import java.util.Scanner;
  2. Crear el Scanner: Dentro del `main`, crea el objeto: Scanner sc = new Scanner(System.in);
  3. Leer datos: Usa los métodos adecuados (nextInt(), nextDouble(), nextLine()).
  4. Cerrar el Scanner: scanner.close(); (para liberar recursos).

Ejemplo completo:

import java.util.Scanner;

public class EjemploScanner {

    public static void main(String[] args) {
        Scanner entrada = new Scanner(System.in);

        System.out.print("¿Cómo te llamas? ");
        String nombre = entrada.nextLine();

        System.out.print("¿Cuántos años tienes? ");
        int edad = entrada.nextInt();

        System.out.println("Hola " + nombre + ", tienes " + edad + " años.");

        entrada.close();
    }
}

Métodos útiles de Scanner: `nextInt()` (para leer enteros), `nextDouble()` (para decimales), `nextLine()` (para texto con espacios).
