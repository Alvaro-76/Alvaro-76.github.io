Entrada y Salida de Datos

En esta sección aprenderemos cómo mostrar información al usuario por consola y cómo leer lo que escribe por teclado.

La entrada y salida de datos en Java se realiza principalmente a través de la consola, utilizando la clase System para la salida y la clase Scanner para la entrada. A continuación, veremos cómo usar ambas para crear programas interactivos.

Mostrando Información: La clase System

Para mostrar información en la consola, usamos de la clase System el método print(). Existen varios métodos en esta clase, que veremos a continuación:

System.out.print()

Para mostrar texto sin salto de línea.

System.out.print("Hola");
System.out.print("Mundo");

El ejemplo mostrará "HolaMundo" en la consola (todo en la misma línea).

System.out.println()

Para mostrar texto con salto de línea.

System.out.println("Hola");
System.out.println("Mundo");

El ejemplo mostrará "Hola" y "Mundo" en líneas separadas.

System.out.printf()

Para formatear la salida de manera precisa (muy útil para números decimales).

System.out.printf("Número: %d Pi: %.2f", 42, 3.14159);

El ejemplo mostrará "Número: 42 Pi: 3.14" en la consola.

Ejemplo completo:

public class EjemploSalida {
    public static void main(String[] args) {
        System.out.println("¡Hola, Mundo!");
        System.out.println("Bienvenido a Java.");
    }
}

Este ejemplo muestra por pantalla el mensaje "¡Hola, Mundo!" seguido del mensaje "Bienvenido a Java." con un salto de línea entre ambos.

El resultado sería:

¡Hola, Mundo!
Bienvenido a Java.

Formatos comunes en printf:

Formato Descripción Ejemplo Salida
%d Entero decimal System.out.printf("Número: %d", 42); Número: 42
%f Número de punto flotante System.out.printf("Pi: %.2f", 3.14159); Pi: 3.14
%s Cadena de texto System.out.printf("Hola, %s!", "Mundo"); Hola, Mundo!
%n Salto de línea (equivale a \n) System.out.printf("Línea 1%nLínea 2"); Línea 1
Línea 2

Obteniendo Datos: La Clase Scanner

Para que nuestros programas interactúen con el usuario, necesitamos leer lo que escribe por teclado. Usamos la clase Scanner, que viene incluida en Java.

¿Por qué usar Scanner?

Imagina que tu programa necesita saber la edad del usuario. No podemos adivinarla, así que necesitamos pedirle que la ingrese. La clase Scanner nos permite hacer exactamente esto: leer lo que el usuario escribe desde el teclado.

Pasos para usar Scanner:

  1. Importar la clase: Añade al principio del archivo (justo después del package): import java.util.Scanner;
  2. Crear el Scanner: Dentro del main, crea el objeto: Scanner sc = new Scanner(System.in);
  3. Leer datos: Usa los métodos adecuados (nextInt(), nextDouble(), nextLine()).
  4. Cerrar el Scanner: sc.close(); (para liberar recursos).

Métodos más utilizados de Scanner:

Método Qué hace Ejemplo
nextLine() Lee una línea completa de texto (incluye espacios) String texto = entrada.nextLine();
next() Lee la siguiente palabra (se detiene en el primer espacio) String palabra = entrada.next();
nextInt() Lee un número entero int numero = entrada.nextInt();
nextDouble() Lee un número decimal double precio = entrada.nextDouble();
nextBoolean() Lee un valor true/false boolean activo = entrada.nextBoolean();
next().charAt(0) Lee un solo carácter char letra = entrada.next().charAt(0);

Ejemplo completo:

import java.util.Scanner;

public class EjemploScanner {
    public static void main(String[] args) {
        // Crear el objeto Scanner para leer desde el teclado
        Scanner entrada = new Scanner(System.in);

        // Pedir el nombre
        System.out.print("¿Cómo te llamas? ");
        String nombre = entrada.nextLine();

        // Pedir la edad
        System.out.print("¿Cuántos años tienes? ");
        int edad = entrada.nextInt();

        // Mostrar el resultado
        System.out.println("Hola " + nombre + ", tienes " + edad + " años.");

        // Cerrar el Scanner (importante para liberar recursos)
        entrada.close();
    }
}
Problema común: Si usas nextInt() seguido de nextLine(), el nextLine() puede quedarse vacío. Esto ocurre porque nextInt() lee el número pero deja el "Enter" en la memoria temporal. Solución: Añade un entrada.nextLine(); extra después de nextInt() para limpiar esa memoria.

Ejemplo resolviendo el problema del Enter:

System.out.print("¿Cuál es tu edad? ");
int edad = entrada.nextInt();
entrada.nextLine(); // ← Importante: limpiar el buffer

System.out.print("¿Cuál es tu nombre? ");
String nombre = entrada.nextLine(); // Ahora sí funciona correctamente
