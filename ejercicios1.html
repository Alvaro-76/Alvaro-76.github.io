Ejercicio 1: Manejo de Cadenas

Fácil

Escribe un programa que pida al usuario una frase y luego muestre la longitud de esa frase, la frase en mayúsculas y la frase en minúsculas.

Pistas:

  • Importa java.util.Scanner.
  • Crea un objeto Scanner asociado a System.in.
  • Usa nextLine() para leer la frase.
  • Usa length() para obtener la longitud de la frase.
  • Usa toUpperCase() y toLowerCase() para convertir la frase.

Solución

Se crea un Scanner, se pide la frase, se lee y se imprime la longitud, mayúsculas y minúsculas.

import java.util.Scanner;

public class ManejoCadenas {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Introduce una frase: ");
        String frase = sc.nextLine();
        
        System.out.println("Longitud de la frase: " + frase.length());
        System.out.println("Frase en mayúsculas: " + frase.toUpperCase());
        System.out.println("Frase en minúsculas: " + frase.toLowerCase());
        
        sc.close();
    }
}

Ejercicio 2: Mayor de edad

Fácil

Crea un programa que pida al usuario su edad y determine si es mayor de edad (18 años o más). Muestra un mensaje adecuado según el resultado.

Si es mayor de edad, muestra "Eres mayor de edad". Si no lo es, muestra "Eres menor de edad".

Solución

Se define un método esMayorEdad que recibe la edad como parámetro y devuelve un booleano. En el main se usa Scanner para leer la edad y se invoca el método.

import java.util.Scanner;

public class MayorEdad {
    public static boolean esMayorEdad(int edad) {
        return edad >= 18;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Ingresa tu edad: ");
        int edad = sc.nextInt();
        
        if (esMayorEdad(edad)) {
            System.out.println("Eres mayor de edad.");
        } else {
            System.out.println("Eres menor de edad.");
        }
        
        sc.close();
    }
}

El nombre de la clase es MayorEdad. Si tienes otro nombre para la clase, asegúrate de que coincida con el nombre del archivo .java.

Ejercicio 3: Semáforo

Media

Escribe un programa que simule un semáforo. El programa debe pedir al usuario el color del semáforo (rojo, amarillo o verde) y mostrar un mensaje correspondiente.

Solución 1

Se usa un if anidado para verificar el color ingresado por el usuario y mostrar el mensaje correspondiente.

import java.util.Scanner;

public class Semafaro {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Ingresa el color del semáforo (rojo, amarillo o verde): ");
        String color = sc.nextLine();
        
        if (color.equals("rojo")) {
            System.out.println("Detente.");
        } else if (color.equals("amarillo")) {
            System.out.println("Prepárate para detenerte.");
        } else if (color.equals("verde")) {
            System.out.println("Pasa.");
        } else {
            System.out.println("Color no válido.");
        }
        
        sc.close();
    }
}

Solución 2 (switch)

Se puede usar un switch para simplificar la lógica de selección del color.

import java.util.Scanner;

public class SemafaroSwitch {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Ingresa el color del semáforo (rojo, amarillo o verde): ");
        String color = sc.nextLine();
        
        switch (color) {
            case "rojo":
                System.out.println("Detente.");
                break;
            case "amarillo":
                System.out.println("Prepárate para detenerte.");
                break;
            case "verde":
                System.out.println("Pasa.");
                break;
            default:
                System.out.println("Color no válido.");
        }
        
        sc.close();
    }
}

El switch es una estructura de control que permite evaluar una variable contra múltiples casos de manera más limpia que varios if-else anidados. En este caso, se evalúa el color ingresado por el usuario y se ejecuta el bloque de código correspondiente a cada color.

Ejercicio 4: Contar números pares

Media

Escribe un programa que pida al usuario un número entero positivo y luego cuente cuántos números pares hay desde 1 hasta ese número (inclusive) y los muestre.

Solución

Se usa un bucle for para iterar desde 1 hasta el número ingresado por el usuario, y se cuenta cuántos de esos números son pares usando el operador módulo y los muestra.

import java.util.Scanner; 
public class ContarPares {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Ingresa un número entero positivo: ");
        int numero = sc.nextInt();
        int contadorPares = 0;   

        for (int i = 1; i <= numero; i++) {
            if (i % 2 == 0) {
                System.out.println(i + " es un número par.");
                contadorPares++;
            }
        }

        System.out.println("Hay " + contadorPares + " números pares desde 1 hasta " + numero);
        sc.close();
    }
}

Solución 2 (while)

También se puede resolver usando un bucle while en lugar de for.

import java.util.Scanner;
public class ContarParesWhile {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Ingresa un número entero positivo: ");
        int numero = sc.nextInt();
        int contadorPares = 0;
        int i = 2; // Empezamos desde el primer número par

        while (i <= numero) {  
            System.out.println(i + " es un número par.");          
            contadorPares++;            
            i += 2; // Incrementar de dos en dos para evitar comprobar si es par
        }

        System.out.println("Hay " + contadorPares + " números pares desde 1 hasta " + numero);
        sc.close();
    }
}

Ejercicio 5: Validación contraseña

Media

Escribe un programa que pida al usuario una contraseña y luego valide si cumple con los siguientes requisitos:

  • Debe tener al menos 8 caracteres.
  • Debe contener al menos una letra mayúscula.
  • Debe contener al menos una letra minúscula.
  • Debe contener al menos un número.

El programa debe mostrar un mensaje indicando si la contraseña es válida o no, y en caso de no ser válida, debe indicar qué requisitos no se cumplen.

Solución 1 (enfoque directo)

En esta solución se utiliza un enfoque directo en el main. Se declaran variables booleanas para cada requisito, se comprueba la longitud y luego se recorre la cadena carácter por carácter actualizando las banderas. Finalmente se muestran los resultados.

import java.util.Scanner;

public class ValidadorPassword {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Introduce una contraseña: ");
        String password = sc.nextLine();
        
        boolean tiene8caracteres = false;
        boolean tieneMayuscula = false;
        boolean tieneMinuscula = false;
        boolean tieneNumero = false;
        
        if (password.length() >= 8) {
            tiene8caracteres = true;
        }
        
        for (int i = 0; i < password.length(); i++) {
            char c = password.charAt(i);
            
            if (c >= 'A' && c <= 'Z') {
                tieneMayuscula = true;
            }
            if (c >= 'a' && c <= 'z') {
                tieneMinuscula = true;
            }
            if (c >= '0' && c <= '9') {
                tieneNumero = true;
            }
        }
        
        System.out.println();
        
        if (tiene8caracteres && tieneMayuscula && tieneMinuscula && tieneNumero) {
            System.out.println("La contraseña es VÁLIDA.");
        } else {
            System.out.println("La contraseña NO es válida. Faltan los siguientes requisitos:");
            if (!tiene8caracteres) {
                System.out.println("   - Debe tener al menos 8 caracteres.");
            }
            if (!tieneMayuscula) {
                System.out.println("   - Debe contener al menos una letra mayúscula.");
            }
            if (!tieneMinuscula) {
                System.out.println("   - Debe contener al menos una letra minúscula.");
            }
            if (!tieneNumero) {
                System.out.println("   - Debe contener al menos un número.");
            }
        }
        
        sc.close();
    }
}

Solución 2 (código compacto)

En esta solución se utiliza un enfoque similar, pero con una estructura más compacta: se actualizan las variables booleanas usando el operador || dentro del bucle y se declara la comprobación de longitud directamente.

import java.util.Scanner;

public class ValidadorPassword {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Introduce una contraseña: ");
        String password = sc.nextLine();
        
        boolean tieneMayuscula = false;
        boolean tieneMinuscula = false;
        boolean tieneNumero = false;
        boolean tiene8caracteres = password.length() >= 8;
        
        for (int i = 0; i < password.length(); i++) {
            char c = password.charAt(i);
            tieneMayuscula = tieneMayuscula || (c >= 'A' && c <= 'Z');
            tieneMinuscula = tieneMinuscula || (c >= 'a' && c <= 'z');
            tieneNumero = tieneNumero || (c >= '0' && c <= '9');
        }
        
        System.out.println();
        
        if (tiene8caracteres && tieneMayuscula && tieneMinuscula && tieneNumero) {
            System.out.println("La contraseña es VÁLIDA.");
        } else {
            System.out.println("La contraseña NO es válida. Faltan los siguientes requisitos:");
            if (!tiene8caracteres) {
                System.out.println("   - Debe tener al menos 8 caracteres.");
            }
            if (!tieneMayuscula) {
                System.out.println("   - Debe contener al menos una letra mayúscula.");
            }
            if (!tieneMinuscula) {
                System.out.println("   - Debe contener al menos una letra minúscula.");
            }
            if (!tieneNumero) {
                System.out.println("   - Debe contener al menos un número.");
            }
        }
        
        sc.close();
    }
}

Solución 3 (con métodos separados)

En esta solución se crean métodos auxiliares para mejorar la modularidad: tieneRango verifica si la cadena contiene algún carácter en un rango dado, y todosCumplidos comprueba que un array de booleanos sea todo true. El main queda más limpio y reutilizable.

import java.util.Scanner;

public class ValidadorPassword {
    
    public static boolean tieneRango(String cadena, char min, char max) {
        for (char c : cadena.toCharArray()) {
            if (c >= min && c <= max) return true;
        }
        return false;
    }
    
    public static boolean todosCumplidos(boolean[] requisitos) {
        for (boolean cumple : requisitos) {
            if (!cumple) return false;
        }
        return true;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        System.out.print("Introduce una contraseña: ");
        String password = sc.nextLine();
        
        String[] mensajes = {
            "al menos 8 caracteres",
            "una letra mayúscula", 
            "una letra minúscula",
            "un número"
        };
        
        boolean[] requisitos = {
            password.length() >= 8,
            tieneRango(password, 'A', 'Z'),
            tieneRango(password, 'a', 'z'),
            tieneRango(password, '0', '9')
        };
        
        if (todosCumplidos(requisitos)) {
            System.out.println("\nLa contraseña es VÁLIDA.");
        } else {
            System.out.println("\nLa contraseña NO es válida. Faltan:");
            for (int i = 0; i < requisitos.length; i++) {
                if (!requisitos[i]) {
                    System.out.println("   - Debe tener " + mensajes[i] + ".");
                }
            }
        }
        
        sc.close();
    }
}

Esta versión es más modular: cada método tiene una responsabilidad única. Además, el uso de arrays permite gestionar fácilmente los requisitos y sus mensajes.
