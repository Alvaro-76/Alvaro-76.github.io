Estructuras de Control

En esta sección aprenderás sobre las estructuras de control en Java, que te permiten dirigir el flujo de tu programa según ciertas condiciones o repetir bloques de código.

Existen tres tipos principales de estructuras de control:

  • Condicionales: Permiten ejecutar bloques de código solo si se cumple una condición.
  • Bucles: Permiten repetir bloques de código mientras se cumpla una condición.
  • Manejo de excepciones: Permite capturar y gestionar errores que pueden ocurrir durante la ejecución del programa.

Condicionales

¿Qué es una estructura condicional? Es una construcción que permite ejecutar diferentes bloques de código dependiendo de si una condición es verdadera o falsa. Esto es fundamental para tomar decisiones en tu programa.

Las estructuras condicionales en Java incluyen:

  • if: Ejecuta un bloque de código si se cumple una condición.
  • if-else: Ejecuta un bloque de código si se cumple una condición, y otro bloque si no se cumple.
  • switch: Permite seleccionar entre múltiples opciones basadas en el valor de una variable.

Ejemplo de uso de if-else:

        
int numero = 10;
if (numero > 0) {
    System.out.println("El número es positivo.");
} else {
    System.out.println("El número es negativo o cero.");
}

En este ejemplo, el programa verifica si el número es mayor que cero y muestra un mensaje en consecuencia.

Ejemplo de uso de if-else if-else:


int numero = 0;
if (numero > 0) {
    System.out.println("El número es positivo.");
} else if (numero < 0) {
    System.out.println("El número es negativo.");
} else {
    System.out.println("El número es cero.");
}

En este ejemplo, el programa verifica si el número es positivo, negativo o cero, y muestra un mensaje adecuado para cada caso.

Importante: Se pueden anidar múltiples estructuras condicionales para manejar casos más complejos, pero es recomendable mantener el código claro y legible para evitar confusiones.

Ejemplo de uso de switch:


int dia = 3;
switch (dia) {
    case 1:
        System.out.println("Lunes");
        break;
    case 2:
        System.out.println("Martes");
        break;
    case 3:
        System.out.println("Miércoles");
        break;
    default:
        System.out.println("Día no válido");
}

En este ejemplo, el programa utiliza una estructura switch para imprimir el nombre del día de la semana basado en el valor de la variable dia.

Nota: Es importante usar la palabra clave break al final de cada caso en un switch para evitar que el programa ejecute casos adicionales de manera no intencionada.

Bucles

¿Qué es un bucle? Un bucle es una estructura de control que permite repetir un bloque de código mientras se cumpla una condición específica. Esto es útil para realizar tareas repetitivas sin tener que escribir el mismo código varias veces.

Los bucles existentes en Java son:

  • for: Repite un bloque de código un número específico de veces.
  • foreach: Repite un bloque de código para cada elemento de una colección o arreglo.
  • while: Repite un bloque de código mientras se cumpla una condición.
  • do-while: Similar a while, pero garantiza que el bloque de código se ejecute al menos una vez.

Ejemplo de uso de for:


for (int i = 0; i < 5; i++) {
    System.out.println("Iteración: " + i);
}

En este ejemplo, el programa imprime las iteraciones del 0 al 4 utilizando un bucle for.

Ejemplo de uso de foreach:


String[] frutas = {"Manzana", "Banana", "Cereza"};
for (String fruta : frutas) {
    System.out.println("Fruta: " + fruta);
}

En este ejemplo, el programa utiliza un bucle foreach para iterar sobre un arreglo de frutas y imprimir cada una de ellas.

Ejemplo de uso de while:


int contador = 0;
while (contador < 5) {
    System.out.println("Contador: " + contador);
    contador++;
}

En este ejemplo, el programa utiliza un bucle while para imprimir el valor del contador mientras sea menor que 5.

los bucle while estan repitiendo el bloque de código mientras la condición sea verdadera.

Ejemplo de uso de do-while:


int contador = 0;
do {
    System.out.println("Contador: " + contador);
    contador++;
} while (contador < 5);

En este ejemplo, el programa utiliza un bucle do-while para imprimir el valor del contador, garantizando que el bloque de código se ejecute al menos una vez, incluso si la condición inicial no se cumple.

Importante: Recuerda que es fundamental evitar bucles infinitos, asegurándote de que la condición de salida se cumpla en algún momento.

Como saber que tipo de bucle usar:

  • for: Cuando conoces de antemano cuántas veces quieres repetir el bloque de código.
  • foreach: Cuando necesitas iterar sobre una colección o arreglo sin preocuparte por los índices.
  • while: Cuando no sabes cuántas veces se repetirá el bloque, pero conoces la condición para continuar.
  • do-while: Cuando necesitas que el bloque de código se ejecute al menos una vez, incluso si la condición no se cumple inicialmente.

Es importante elegir el tipo de bucle adecuado según la situación específica para mantener el código claro y eficiente.

Control de Flujo

¿Qué es el control de flujo? El control de flujo se refiere a la forma en que se dirige la ejecución de un programa. Las estructuras de control, como las condicionales y los bucles, son herramientas fundamentales para controlar el flujo de tu programa y tomar decisiones basadas en condiciones específicas.

Pero estos flujos pueden ser alterados por las palabras clave break, continue y return, que permiten modificar el comportamiento normal del flujo de ejecución de un programa.

El control de flujo es esencial para crear programas dinámicos y flexibles que puedan adaptarse a diferentes situaciones y datos de entrada. Al dominar las estructuras de control, podrás escribir código más eficiente y efectivo, capaz de manejar una amplia variedad de escenarios.


// Ejemplo de uso de break
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;
    }
    System.out.println(i);
}

// Ejemplo de uso de continue
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        continue;
    }
    System.out.println(i);
}

// Ejemplo de uso de return
public static int sumar(int a, int b) {
    if (a < 0 || b < 0) {
        return -1; // Devuelve -1 si alguno es negativo
    }
    return a + b;
}

En el primer ejemplo, el bucle se detiene cuando i es igual a 5 debido al uso de break.

En el segundo ejemplo, el número 5 se omite en la impresión debido al uso de continue.

En el tercer ejemplo, la función sumar devuelve -1 si alguno de los parámetros es negativo, utilizando return para salir de la función.

Importante: El control de flujo es una parte fundamental de la programación, y su comprensión es clave para desarrollar programas robustos y eficientes. Asegúrate de practicar y experimentar con diferentes estructuras de control para fortalecer tus habilidades de programación.
