<h1>Aprendiendo Java – Proyecto Educativo</h1>

<p>
Este repositorio contiene una web educativa creada como material de apoyo durante mi aprendizaje de Java.
Aunque comenzó como apuntes personales, el objetivo es que cualquier persona que quiera aprender pueda utilizarla como recurso estructurado y práctico.
</p>

<hr>

<h2>Objetivo del Proyecto</h2>

<p>
El propósito principal de esta web es documentar el proceso de aprendizaje siguiendo el mismo orden del curso recibido,
organizando los contenidos de forma clara y progresiva.
</p>

<p>
Está pensada para:
</p>

<ul>
<li>Estudiantes que comienzan en Java.</li>
<li>Personas que quieran reforzar conceptos básicos.</li>
<li>Quienes prefieren aprender mediante teoría acompañada de ejercicios prácticos.</li>
</ul>

<hr>

<h2>Estructura del Contenido</h2>

<p>El sitio está organizado por temas, siguiendo el orden original del curso:</p>

<ul>
<li>Lecciones teóricas estructuradas por unidades.</li>
<li>Ejercicios prácticos al final de cada bloque.</li>
<li>Soluciones incluidas (en algunos casos con más de una posible implementación).</li>
</ul>

<p>
Los ejemplos están enfocados en fundamentos del lenguaje: variables, estructuras de control,
métodos, clases y principios básicos de programación orientada a objetos.
</p>

<hr>

<h2>Versión de Java</h2>

<p>
El contenido sigue un curso práctico estándar de fundamentos de Java.
No se especifica una versión concreta del lenguaje, ya que los ejemplos utilizados corresponden a sintaxis básica
compatible con versiones modernas del entorno Java estándar.
</p>

<hr>

<h2>Enfoque Técnico de la Web</h2>

<ul>
<li>Proyecto desarrollado como sitio web estático.</li>
<li>Estructurado en archivos HTML independientes por tema.</li>
<li>Uso de CSS para el diseño visual.</li>
<li>Uso de JavaScript para pequeñas funcionalidades dinámicas.</li>
</ul>

<p>
La publicación se realiza mediante <strong><span>GitHub Pages</span></strong>, utilizando el servicio de la plataforma <strong>:contentReference[oaicite:0]{index=0}</strong>.
</p>

<hr>

<h2>Metodología</h2>

<p>
El contenido sigue exactamente el orden del curso original recibido.
La explicación se complementa con ejemplos prácticos y ejercicios resueltos.
</p>

<p>
Parte del diseño visual y parte del contenido han sido generados con apoyo de inteligencia artificial,
utilizada como herramienta de asistencia, no como sustituto del aprendizaje.
</p>

<hr>

<h2>Uso del Proyecto</h2>

<p>
Este material puede utilizarse libremente como recurso de aprendizaje personal.
Se recomienda practicar los ejercicios antes de consultar las soluciones para aprovechar mejor el proceso formativo.
</p>

<hr>

<h2>Finalidad</h2>

<p>
Este repositorio forma parte de un proceso continuo de formación en programación.
El objetivo no es presentar un curso oficial, sino un recurso práctico,
documentado y accesible para reforzar conocimientos en Java.
</p>

<hr>

<p>
Si estás aprendiendo Java, puedes utilizar esta web como apoyo.
La mejor forma de aprender programación es escribir código, equivocarse y volver a intentarlo.
</p>
